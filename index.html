CSS：

在头部（head）添加 style 标签，即可添加CSS


比如说，在style中添加
  • h1 {
    color: blue
    font-size: 11px
    }

这样可以把h1的文字颜色改为蓝色，并且也将大小改为11px（px=像素）。
